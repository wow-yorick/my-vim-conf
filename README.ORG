VIM配置FOR(php go)

========================
基于Vundle进行包管理

* 资源地址
** vim官方包 [[https://www.vim.org/download.php][官方下载地址]]
** vim vundle包管理工具地址 https://github.com/VundleVim/Vundle.vim

* 安装
1. 确保安装了git curl 工具
2. 安装插件 打开vim  运行 ~:PluginInstall~

* 自动安装脚本
:: 运行 ./install.sh 

* 说明
* .vimrc为配置文件
* .vim 为依赖的插件
